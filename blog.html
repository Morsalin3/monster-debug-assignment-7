1. Difference of Local Starage and Session Storage ?

ans :sessionStorage is similar to localStorage ; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

2. Difference or Global Scope and Block Scope ?

ans : Global Scope: When a variable is declared inside a function, it is only accessible within that function and cannot be used outside that function.

Block Scope: A variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.

4. How Many Ways Can We Get Undefind ?

ans : Eight Ways to Get Undefined.
1. Variable that is not initialized will give undefined.
2. Function with no return.
3. Parameter that is not passed will be undefined.
4. If return has nothing on the right side will return undefined.
5. Property that doesn't exists on an object will give you undefined.
6. Accessing array elements outside of the index range.
7. Deleting an element inside an array.
8. Set a value directly to undefined.
